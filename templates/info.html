Github Pull Requests Status

Please access the api via the below format to get the status of your pull requests 


        hostname/:owner/:repo/:token
Token is your personal token generated from GitHub, this is required to avoid Github's api request limit.

The below details will provide you a overview of the status

  • REVIEWER_PENDING - PR created and reviewer is not assigned
  • REVIEW_IN_PROGRESS - Reviewer is assigned and PR is not yet approved
  • STATUS_CHECK_PENDING - PR is approved and status checks are not reported
  • FAILING - Status checks are failing for the PR
  • MERGE_PENDING - PR is approved and status checks have passed

The HTTP endpoint which returns the open PR states as json:

{
  prs: [
    {
      "id": 1,
      "state": "REVIEWER_PENDING"
    },
    {
      "id": 2,
      "state": "FAILING"
    }
  ]
}
